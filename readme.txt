== Description ==

This **Extra Options Addon for Bookster** allows you to add extra services to your booking forms to collect additional payments from customers.

**NOTE: [Bookster 2.0.0 or later](https://wordpress.org/plugins/bookster/) is required to use this plugin.**

== Features ==

- **Create Optional Service**: Add descriptive names and images to your extra options to clearly communicate their value to customers.
- **Set Custom Pricing**: Assign specific prices or additional charges to each extra option, allowing you to increase revenue or offer premium services.
- **Customize Duration**: Set the duration for each extra option to accurately reflect the time required for service.
- **Targeted Assignments**: Assign extras to specific services or categories, ensuring that only relevant options are presented to customers.
- **Quantity Selection**: Enable customers to choose multiple extras in one go by allowing them to specify the quantity for each option.

== Installation ==
Manual installation is easy and takes less than one minute.

1. Search for "Bookster Extra Options" and install this plugin.
2. Activate the plugin through the ‘Plugins‘ menu in WordPress.
3. Go to the **Bookster** > **Addons** menu to set up your extra services!

Enjoy!
